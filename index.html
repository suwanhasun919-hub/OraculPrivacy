Privacy Policy

Last updated: April 1, 2026

Oracul is a mobile application designed for match analytics, providing insights, statistics, and key match events in real time.

Data Collection

Oracul does not collect, store, or process any personal data. We do not require user registration and do not request names, email addresses, phone numbers, or any other personally identifiable information.

The application does not use analytics services (including Firebase), does not display advertisements, and does not use advertising identifiers.

Third-Party API

The application retrieves match analytics data from a third-party API. Statistics and event details are loaded directly from the provider’s servers. The application does not store this information and does not associate it with any individual user.

Data Sharing

Since no personal data is collected, no data is sold, shared, or transferred to third parties.

Children’s Privacy

The application is not specifically directed toward children under the age of 13. We do not knowingly collect any data from children.

Security

Because the application does not collect or store personal information, there are no risks related to user data storage or processing within the app.

Политика конфиденциальности

Последнее обновление: 1 апреля 2026

Oracul — мобильное приложение для аналитики матчей, предоставляющее статистику, инсайты и ключевые события в режиме реального времени.

Сбор данных

Приложение Oracul не собирает, не хранит и не обрабатывает персональные данные пользователей. Регистрация не требуется. Мы не запрашиваем имя, email, номер телефона или другие персональные данные.

Приложение не использует аналитические сервисы (включая Firebase), не показывает рекламу и не применяет рекламные идентификаторы.

Сторонний API

Для получения аналитических данных используется сторонний API. Информация загружается напрямую с серверов поставщика. Приложение не сохраняет эти данные и не связывает их с пользователями.

Передача данных

Поскольку персональные данные не собираются, они не передаются и не продаются третьим лицам.

Дети

Приложение не предназначено специально для детей младше 13 лет. Сбор данных о детях не осуществляется.

Безопасность

Так как персональные данные не собираются и не хранятся, риски, связанные с их обработкой, отсутствуют.
